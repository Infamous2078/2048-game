Rapport du TP2

Choix de présentation que nous avons fait
  • Nous avons décider d'utiliser une table html pour le jeu car nous avons penser que ceci faciliterai le placemement des tuiles lors de l'execution de l'algorithme en cherchant les cases avec leur index
  • Nous avons opter pour l'utilisation d'un tableau dynamique javascript pour calculer le placement des tuiles puis ensuite affecter cela à la table html
Ce que nous avons appris
  • L'utilisation de jQuery qui est beaucoup plus simple, intuitif et plus courte a utiliser que javascript natif comme par exemple pour changer le contenu html d'un élément
  • Le changement de style à partie de javascript, pendant l'exécution du code
  • L'ajout de balise au DOM pendant l'exécution avec javascript et jQuery
